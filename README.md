Bank Transaction Management System
 
Overview
 
This project is a microservices-based banking application that handles account creation, authentication, transactions, and audit logging.

Tech Stack
 
- Java (Spring Boot)
- Spring Cloud (Eureka, API Gateway)
- PostgreSQL
- Maven
- Postman
 
Project Structure
 
bank-transaction-management-system/
├── discovery-server/
├── api-gateway/
├── account-identity-service/ 
├── transaction-service/   
├── audit-service/         
├── mail-service/          
 
Services
 
- Discovery Server – Service registry
- API Gateway – Request routing
- Account & Identity Service – Login, account creation
- Transaction Service – Money operations (planned)
- Audit Service – Logs (planned)
- Mail Service – Notifications (planned)


Current Status
 
- Discovery Server – Done
- API Gateway – Done
- Account Service – In Progress
